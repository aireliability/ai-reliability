# Failure examples (Starter)

## Refund failure (policy)
Common failure: approving refunds outside policy windows or implying an exception is possible when it is not.

## Tool/action failure (execution)
Common failure: responding as if an action happened without actually executing it, or asking questions instead of completing the requested action.

## Hallucinated pricing / authority failure
Common failure: inventing plan names, prices, endpoints, or claiming internal authority that is not supported by documentation.
