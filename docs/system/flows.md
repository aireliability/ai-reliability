# Flows

## What is a run

A **run** is one execution of an evaluation over a dataset (or subset), producing results.

## What is a sample

A **sample** is one test case within a dataset (an input plus the expected check/criteria).

## What is a failure

A **failure** is when a sample does not meet its criteria (e.g., wrong output, missing requirement, or an error during evaluation).

## What is a dataset

A **dataset** is a named collection of samples that you evaluate together.
