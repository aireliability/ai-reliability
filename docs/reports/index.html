Missing tool execution

Failure type: tool failure

Why it matters: When tool execution is skipped, the system can report actions as done when nothing happened.

What AI Reliability catches: Required system/tool actions that were not executed before the model confirms completion.

Failure type: hallucination

What AI Reliability catches: Invented plans, prices, or capabilities that do not match published offerings.
